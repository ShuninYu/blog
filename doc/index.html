blog编写格式参考文档

2024-6-20 - 4:30 | 最后更新：年-月-日

为了方便我编写blog，特意编写了这个用于本地的参考文档页面。
因为blog都是纯手敲html页面，所以使用空白模板以后想要添加特殊内容时就会比较麻烦。
使用ctrl + f 搜索需要的内容，或者使用右侧导航栏快速跳到需要的部分。
由于主要是用于本地的文档，所以本页面不带有左侧的网站导航。

段落标题

        <h2 id=""></h2>

在id项的双括号内填写右侧导航按钮识别用的id，要与导航栏button元素内的location.href项井号后内容相同。
在h2元素内填写标题内容，要与导航栏button元素内文字相同。

代码

                            <div>
    <pre>
        <code>代码内容第一行
代码内容中间行
代码最后一行</code>
</pre>
</div>

新建一行粘贴后，让左侧紧贴边缘！因为VSCode会自动添加缩进导致代码底部错误的多出很多空间。
如果预览的时候高亮错误，可以在code元素中添加class来标注代码类型，例如：class="Bash" 来标注Bash代码，插件会自动识别并高亮显示。
使用 &lt; 来代替代码以及正文中的 <
使用 &amp; 来代替转义字符中的 &
代码不能对齐html格式的缩进，否则会显示出错。
首行必须在code标签后面，末行必须在/code标签前面，代码每一行都必须按代码实际内容缩进。
最后一个pre标签必须取消缩进，否则代码块和其他文字的间距会显示错误。（详细看示例）
blog使用jquery，代码段用上面的格式插入，插件会自动识别代码类型。

示例代码：

                            <div>
    <pre>
        <code>using System;
        namespace HelloWorldApplication
        {
            /* 类名为 HelloWorld */
            class HelloWorld
            {
                /* main函数 */
                static void Main(string[] args)
                {
                    /* 我的第一个 C# 程序 */
                    Console.WriteLine("Hello World!");
                    Console.ReadKey();
                }
            }
        }</code>
</pre>
</div>

显示效果

文字超链接

                        <a rel="noopener noreferrer" target="_blank" class="wordbutton" href=""></a>

在a标签中间填写作为超链接高亮的文字。
在href项的双括号里填写点击高亮文字后要打开的链接。

正文中插入文字超链接示例。

普通图片

        <div class="imgbox">
    <img src="">
    <p></p>
</div>

插入的图片都是由一个独立的div包裹着并且居中显示的。
在img元素的src项内填入图片路径。
在p元素内填写图片描述。
如果添加的图片没有描述，就把p元素删掉。

插入图片以及图片描述示例（不含描述的图片将不会有这行文字）

像素图片

        <div class="imgbox pixel">
    <img src="" style="width:;">
    <p></p>
</div>

在img元素的src项内填入像素图片路径。
在img元素的style项内填入像素图片缩放后的尺寸。
在p元素内填写图片描述，同样如果没有描述就删除p元素。
像素图单独拿出来是因为使用特殊的缩放采样效果可以节省文件体积，让像素图可以使用尺寸最小的原图。

像素图效果预览

划掉的文字

        <del></del>

在del元素内填写要被划掉的文字，这个很好理解吧。

添加划掉的文字的示例

黑幕文字

        <span class="heimu" title=""></span>

*使用的是萌娘百科版本的文字黑幕。
在span元素内填写要被黑幕遮住的文字。
在title项内填写指针悬浮在黑幕文字上时，浮现的小窗口内显示的文字。
目前我还没有为黑幕文字做暗色模式的适配，所以显示效果会有点融入背景。

添加被黑幕盖住的文字的示例

备注文字（p.s.）

        <ps></ps>

在ps元素内填入备注文字。
对我自己来说，添加备注文字时要新起一行。
备注文字可以在正文p元素内直接加入。

正文内容中间可以插入备注文字。
备注文字的示例

补充文字

        <div class="intro"></div>

在正文中添加和这段文本以及文章简介一样的文字框。
在<div>标签中间像正文一样使用<p></p>插入文字。

折叠窗口

        <div class="folder folding">
    <button class="foldButton">
        <div class="triangle"><p2>▶</p2><h1></h1></div>
    </button>
    <div class="foldcontent">
    </div>
</div>

向下折叠展开的窗口。
在<h1></h1>内添加折叠窗口标题。
在 "foldcontent" div内像正文一样添加内容。

展开后的文字内容示例

导航栏折叠窗口

        <div class="sideFolder sideFolding">
    <div class="sideFoldHeader">
    <button class="sideFoldButton sideTriangle"><p2>▶</p2></button>
        <button class="sideFoldRightButton" onclick="location.href='#'"></button>
    </div>
    <div class="sideFoldcontent">
        <button class="sideRightButton" onclick="location.href='#'"></button>
    </div>
</div>

侧边栏导航按钮折叠版。
在 ▶ 右侧填入父导航按钮内容，在sideFoldcontent内添加子按钮。
